# 1. 街頭演説(マイク納め)の感想

- JCPサポーターに若い人が多かった。２０代から３０代。
- 「そうだ！」という掛け声が聞かれた
- 演説が終わっても会場に聴衆が大勢残っていた

演説を聞きに来る人たちは声を上げたいんだと感じました。残っていた聴衆やJCPサポータの会話のコーナーがあったら良かったなと思いました。

聴衆を煽るのは良くないですが、聴衆が声を上げていると通行人も盛り上がっているなと思うのではないでしょうか

「日本共産党が政権を取ったら」とか「田村智子を総理大臣に」というキャッチフレーズを織り交ぜて「強さ」をアピールしてほしかった。

# 2. 共産党の政策の感想

## 2.1. 他党にはない政策を言って欲しかった

- 消費税減税の政策は多党も言い出して埋もれてしまった感がありました。他党が言わない政策を全面にだしたほうが良かったなという感想
- TAXTheRichは他党には言えない政策なのでこっちを全面にして欲しかった。ただし横文字は嫌いなので日本語にしてほしい。
- 賃金を上げろという政策には、もうちょっと言い方がないかなと感じました。

## 2.2. わかりやすいキャッチコピーの私案

- 人が死なない、人を殺さない、人が殺されない社会を作る
- 犯罪に巻き込まれない、犯罪を起こさせない、犯罪を見逃さない、犯罪の手助けをしない社会を作る
- 生きていて良かったと思える社会を作る
- 働いていて良かった、働きがいがある職場を作る
- 人にとやかく言われない、人をとやかく言わない社会を作る

## 2.3. 富裕層向け政策

お金に余裕があっても文化が豊かではないと余生を楽しむ機会もないと思います。テレビもエンタテインメントとして低俗化していますし、テレビドラマや映画もレベルが下がっていることは否めないと思います。

- 日銀を正常化して政策金利５％にして海外旅行に行きやすくする
- ６５歳以上に適用されていたマル優の復活
- 国立の文化施設を守る

    国立劇場が閉鎖されていることを知らない国民が多い
