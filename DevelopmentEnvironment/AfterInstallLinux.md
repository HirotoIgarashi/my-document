# Linuxをインストールした後に行う事

## SHELLを変更する

SHELLをbashからfishに変更する。

## ターミナルを変更する

cntl + alt + t で起動するターミナルをXfce 端末からwezterに変更する。

## PATHの設定

PATHに.local/binを追加する

## タッチパッドの有効/無効

タッチパッドの有効/無効を設定するコマンドを作成してキーボードショートカット ctrl + alt + @ に割り当てる。
